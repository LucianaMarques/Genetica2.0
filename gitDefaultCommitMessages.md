Git default commit messages
---------------------------

- **[FCT]** Description 
    *(New functionality added)*
- **[BUG.xx]** Bug description (When solved) 
    *(For bugs – xx: Bug number in the issue tracker)*.
- **[WIP]** Description 
    *(Work in Progress – Lacks testing)*
- **[UPD]** Updated to version 1.x.y
    *(New Release – Can go to production)*
- **[DOC]** Added doc
    *(New Documentation)*
- **[QLTY]** Description
    *(Code refactoring and general beauty parameters)*
- **[OPT]** Description
    *(Optimizations – The code has been optimized)*
